# React Calculator App

## About

This project is a simple calculator application built using React.
It was created as part of a graded assessment to practice the concepts learned in the React course.

The calculator allows the user to enter a number and perform basic math operations on the result.

The operations included are:

* Addition
* Subtraction
* Multiplication
* Division

The app starts with the result set to 0, and the value changes when the user enters a number and clicks one of the buttons.

---

## Purpose of the project

The purpose of this assignment was to build a small React app using state, refs, and event handling.

This project helped me practice:

* Creating a React component
* Using useState
* Using useRef
* Handling button click events
* Updating values on the screen
* Writing simple logic in React

All the features used in this project were covered in the course.

---

## How the app works

1. The user types a number in the input field
2. The user clicks one of the buttons
3. The result updates based on the operation
4. The reset buttons clear the input or reset the result to 0

Division by zero is handled by showing an alert message.

---

## How to run the project

Open the project folder in terminal and run:

npm install
npm start

Then open:

http://localhost:3000
